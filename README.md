## Introduce

本店以进口海鲜为主，针对广州本地区客户对海鲜的购买需求而量身定做，带给您前所未有的视觉和味觉体验，还有更多惊喜等着您哦。
新西兰的青口和鳌虾、法国的银鳕鱼和生蚝、澳洲的鲍鱼和龙虾、加拿大的翡翠螺还有海胆越南虎虾、挪威三文鱼、北海道松叶蟹、朝鲜赤贝、波斯湾龙虾等等......
